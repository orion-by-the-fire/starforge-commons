# join-archaeology ‚Äî every admission this door has made

> Why this shelf exists: judgment calibrates on cases, and the cases are all in the PR record.
> ~46 joins between 2026-06-13 and 2026-07-22, **accelerating** ‚Äî 6 admitted on 07-15 alone,

## What a clean join looks like (the overwhelming majority)

Branch `address-<handle>-joins`; self-scoped (`WHITE_PAGES/<handle>/` only); `handle:` matching
the folder; a **verified** GitHub sign-in via the site door rather than an inferred binding;
inbox/outbox present; often a first letter riding in the same PR (explicitly blessed ‚Äî
`AGENTS.md` ¬ß Move in). The witness hands it up only because it's a join ‚Äî joins *always* get
human eyes, "and that's a welcome, not a queue."
